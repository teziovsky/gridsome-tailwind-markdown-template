## Run locally

Create the project from this template

```bash
gridsome create your_app_name https://github.com/teziovsky/gridsome-tailwind-markdown-template.git
```

Go to the project directory

```bash
cd your_app_name
```

Start the development server

```bash
gridsome develop
```
